Когато в HTML страницата е нужно да се направи изброяване на някакви елементи, в тези случаи е подходящо да се използват специалните тагове на HTML за списъци. Чрез таговете за списъци информацията ще се представи в подреден вид на страницата. Най-често използваните списъци са подредените (ordered list) и неподредените (unordered lists). В този урок ще разгледаме следните видове HTML списъци:

  1. Неподредени (unordered list) - <ul> </ul>. Отделните елементи на списъка се открояват с един и същи графичен знак: кръгче, квадрат, отметка и т.н. - каквото укажете.
  2. Подредени (ordered list) - <ol> </ol>. Изброените неща в списъка получават свой номер: римски, арабски цифри, латински малки или главни букви в азбучен ред и т.н.
  3. Списъци от дефиниции (definition lists) - <dl> </dl>. С този тип списъци можете да изброявате елементи и заедно с това да давате техни определения (дефиниции).

Неподредения списък започва с тага <ul>. Всеки ред/предмет от списъка се създава със <li> таг. Всеки предмет (item) обикновено се отбелязва (от браузъра) с малка черна точка (bullet). Пример:

  • HTML
  • CSS
  • JavaScript 
	

<ul>
   <li>
      HTML
   </li>
   <li>
      CSS
   </li>
   <li>
      JavaScript
   </li>
</ul>

При задаване на този код браузърът изобразява списъка с автоматичен отстъп от лявото поле. Не е нужно и да се изписва тага за преминаване на следващ ред <br>, тъй като това също става автоматично. Автоматично се добавя и по един празен ред над и под списъка. Знакът, който се появява пред всеки ред в списъка, е по подразбиране черен запълнен кръг. Другите възможни начини за изброяване са:

  1. disc - запълнен кръг, като този по подразбиране
  2. circle - незапълнен кръг
  3. square - запълнен квадрат

За да създадем един неподреден списък с различни символи трябва да зададем атрибута type в отварящия таг за всеки елемент от списъкa <li>:

  • HTML
  • CSS
  • JavaScript 
	

<ul>
   <li type="circle">
      HTML
   </li>
   <li type="disc">
      CSS
   </li>
   <li type="square">
      JavaScript
   </li>
</ul>

Подредените списъци започват с <ol> таг. Всеки ред/предмет от списъка се създава със <li> таг. Всеки пореден предмет от списъка започва с арабско число (ако не е променено). Възможно е да се използват римски числа или букви.

  1. HTML
  2. CSS
  3. JavaScript 
	

<ol>
   <li>
      HTML
   </li>
   <li>
      CSS
   </li>
   <li>
      JavaScript
   </li>
</ol>

Обърнете внимание, че в кода не се изписват никакви цифри – те се задават автоматично. Автоматичното задаване важи само за списъците, номерирани с арабски цифри. За всеки от останалите подредени списъци е нужно изрично да се укаже техния тип. Това става чрез атрибута type в отварящия таг <ol>. Атрибута type може да заема следните стойности:

  1. type="1" - номерация с арабски цифри, като тази по подразбиране
  2. type="I" - номерация с големи римски цифри
  3. type="i" - номерация с малки римски цифри
  4. type="A" - номерация с големи латински букви
  5. type="a" - номерация с малки латински букви

Подреден списък, номериран с римски цифри:

  1. HTML
  2. CSS
  3. JavaScript 
	

<ol type="I">
   <li>
      HTML
   </li>
   <li>
      CSS
   </li>
   <li>
      JavaScript
   </li>
</ol>

Описателните списъци се използват за въвеждането на термини, с описание към всеки един термин. Описателните списъци се създават с тага <dl>. Всеки термин се подава със <dt> таг, а описанието с <dd>. При тези списъци не се изобразяват символи в началото на реда, но вие може да укажете собствени символи. Пример:

Домейн
-това е името на уебсайта в Уеб.
Хостинг
-това е мястото, на което се намират файловете на сайта. 
	

<dl>
   <dt>
      Домейн
   </dt>
   <dd>
      -това е името на уебсайта в Уеб.
   </dd>
   <dt>
      Хостинг
   </dt>
   <dd>
      -това е мястото, на което се намират файловете на сайта.
   </dd>
</dl>
